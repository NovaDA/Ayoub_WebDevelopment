BRIGHTER MINDS

Brighter Minds is a platform where the users can keep track of different habits that can change the mantal state of the subject. The user will need to own an account in brighter minds before starting using the application. Once the user has an own account it will be asked how she/he feels and prompted with different questions about how psychologically thus feels at that certain point of the day. The user after the questionnaire will be able to see how many users are feeling the same way depending on the attribute they want to check. All the other users will always stay anonymous the user details will never be stored in the app.

ROLE IN THE PROJECT

Duration : 1 week

Team: 4 members

In the project I did work fullstack focusing on the backend of the app and helping to resolve any issue from the front-end when accessing the data in the backend. I managed to create a database using Flask integrated postgresql. In the database I stored entries required in the app. I created urls to access the required data, i created some block of codes that managed the process of filtering all the data from the DB tables and return only the essential information.
